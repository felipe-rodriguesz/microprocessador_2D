# Simulação Térmica 2D — Microprocessador com Hotspot

## Como rodar
1. Entre na pasta scripts/
2. Execute: bash run_experiments.sh
3. Abra MATLAB em plots/matlab/ e rode plots_comparacao.m

## Esquemas implementados
- FTCS (explícito, 1ª ordem)
- Backward Euler (implícito, 1ª ordem)  
- Crank-Nicolson (implícito, 2ª ordem)
